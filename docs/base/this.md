# this与词法作用域

##  JavaScript 闭包

闭包 是一个函数能够记住并访问其词法作用域的能力，即使该函数在其词法作用域之外执行。

词法作用域（Lexical Scoping）：函数在定义时就确定了它可以访问哪些变量，而不是在执行时。

2. 闭包的关键特性
记忆外部变量：闭包会“记住”它创建时所处的环境

保持变量存活：即使外部函数已经执行完毕，其内部变量也不会被垃圾回收

私有性：可以创建私有变量

## JavaScript 中的 this 指向
this 是 JavaScript 中一个特殊的关键字，它指向当前执行上下文中的对象。this 的值不是静态的，而是根据函数的调用方式动态决定的。

1. 默认绑定（独立函数调用）
在非严格模式下，this 指向全局对象（浏览器中是 window，Node.js 中是 global）
在严格模式下，this 是 undefined

2. 隐式绑定（方法调用）
函数作为对象的方法被调用时，this 指向调用该方法的对象

3. 显式绑定（call/apply/bind）
使用 call()、apply()、bind() 可以显式地设置 this

4. new 绑定（构造函数调用）
使用 new 关键字调用构造函数时，this 指向新创建的实例

5. 箭头函数中的 this
箭头函数没有自己的 this，它继承自外层作用域的 this
